# 構造（設計図）

このテンプレートを使用するプロジェクトの概要レイアウトです。

```
remotion-studio/
  apps/
    _template/        # 基本テンプレート
    3D-template/      # 3Dテンプレート（Three.js; R3Fはオプション）
  packages/           # オプションの共有パッケージ
  scripts/            # CLIスクリプト（create-project、ランナーなど）
  docs/               # リファレンスと運用ノート
```

詳細については、`docs/architecture.md` も参照してください。
